# 🔐 AI-Based Unauthorized WiFi Device Detection System

## 📌 Overview
This project is developed to detect unauthorized and suspicious devices connected to a WiFi network using AI-based analysis and real-time network scanning.

## 🚀 Features
- Real-time WiFi device scanning
- Unauthorized device detection
- AI-based risk score analysis
- Security siren alert system
- Logs and PDF report generation
- Interactive dashboard using Streamlit

## 🛠 Technologies Used
- Python
- Streamlit
- Machine Learning
- Scapy
- Pandas
- Matplotlib
